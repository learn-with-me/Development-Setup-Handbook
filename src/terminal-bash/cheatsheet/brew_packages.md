# Cheatsheet - Brew packages

## cmatrix

Shows matrix screen on terminal

```sh
$ brew install cmatrix
$ cmatrix
```

## asciiquarium

```sh
$ brew install asciiquarium
$ asciiquarium
```

## toilet

Turn any text into ascii character art

```sh
$ brew install toilet
$ toilet <text>
```

## tetris

```sh
$ brew install samtay/tui/tetris
$ tetris
```
