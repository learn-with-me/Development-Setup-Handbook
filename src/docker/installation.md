# Installation

```sh
# Option 1
$ brew cask install docker
$ brew cask reinstall docker

# Option 2
# Download OS specific version and install
https://docker.com/getdocker
# OS-specific instructions. For production
https://docs.docker.com/engine/installation

# Installation Location: /usr/local/bin
```

## Verify Installation

```sh
# Display installed version number
$ docker --version

# Display more details version information with client and server
$ docker version

# Display system-wide information
$ docker info

# Docker Compose version
$ docker-compose --version
```
