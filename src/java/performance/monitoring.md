# Production Monitoring

* Operations activity
* Collect and aggregate metric data
* Metric data storage
* Analysis, visualization and alerting

Production monitoring is broken into three parts

* Collect
    * Dropwizard metrics
    * Micrometer metrics
    * Netflix Spectator
    * Google OpenCensus
    * Prometheus Lib
* Store
    * InfluxDB
    * ElasticSearch
    * Graphite
    * OpenTSDB
    * Prometheus TSDB
    * Other commercial tools like DataDog, etc
* Analyze/Visualize
    * Grafana
    * Kibana
    * JMX tools
    * Prometheus dashboards
